# Домашнее задание к лекции «Web-scrapping»Попробуем 
Скрипт получает интересующие нас статьи на [Хабре](https://habr.com). Парсить страницу со свежими статьями ([вот эту](https://habr.com/ru/all/)) и выбирает те статьи, в которых встречается хотя бы одно из ключевых слов. Эти слова определяются в начале скрипта. Поиск ведется по полному тексту статьи.
